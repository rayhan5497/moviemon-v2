# Contributing

Thanks for your interest in contributing to MovieMon.

## How to Contribute

1. Open an issue to discuss the change or feature.
2. Fork the repository and create a feature branch.
3. Keep changes focused and aligned with the architecture rules.
4. Ensure linting and tests pass before submitting.
5. Open a pull request with a clear description of the changes.

## Code Style

* Follow the established feature-based architecture.
* Avoid cross-feature imports.
* Keep shared utilities in `shared` and feature logic in `features`.

## Reporting Issues

Provide steps to reproduce, expected vs actual behavior, and any logs or screenshots that help diagnose the issue.
